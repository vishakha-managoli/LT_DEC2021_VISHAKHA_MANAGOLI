Webinars

Hacktoberfest

On the occasion of Hacktoberfest starting from 1st October, a month-long celebration of open source software CodeLearners is providing a great opportunity for students to improve their competitive coding skills and also to start making open source contributions and get a chance of working in a community.

Introduction to Git and GitHub

The session will include an introduction to Git & GitHub and a hands-on session explaining basic git commands and the use of GitHub platform while making open source contributions and projects.

Getting Started with Open Source

This webinar was conducted by Mr. Prathamesh Chavan, where he guided us on how one can get started with making contributions in both technical and non-technical fields. This beginner-friendly session also covered the ways to explore open-source with CodeLearners and other great opportunities.

Session on Web Development

The speaker for this session was Ankitkumar Singh. The speaker promoted open-source software development. In this session, he guided students on how to get started with web development..

Workshop on ML and AI in open source
